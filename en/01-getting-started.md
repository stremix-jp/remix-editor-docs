# Getting Started

This guide explains the basic usage of remix-editor.

## Opening the Application

1. Access the remix-editor URL in your browser
2. On first launch, an empty pattern will be created

## Basic Workflow

### 1. Load an Audio File

1. Drag and drop an audio file onto the audio area in the **footer**
2. Or click to open the file selection dialog

Supported formats: MP3, WAV, M4A, OGG, FLAC, AAC (max 100MB)

### 2. Create a Pattern

1. Click the "+" button in the **Pattern List** on the left panel
2. A new pattern will be created and automatically selected
3. Double-click the pattern name to rename it

### 3. Edit the Curve

1. Click on the **Curve Editor** (center canvas) to add points
2. Drag points to move them
3. Select multiple points for batch editing

See [Curve Editing](./04-curve-editing.md) for details.

### 4. Connect a Device (Optional)

1. Launch [Intiface Central](https://intiface.com/central/)
2. Click "Connect" in the **Device Panel** in remix-editor
3. Scan and connect to devices

See [Device Connection](./06-devices.md) for details.

### 5. Play and Preview

1. Press Space or the play button to start playback
2. If devices are connected, they will be controlled in real-time
3. Playback speed can be adjusted

### 6. Save the File

1. Select **File > Save Remix...** from the menu
2. The file will be saved in Remix JSON format

See [File Operations](./08-file-operations.md) for details.
