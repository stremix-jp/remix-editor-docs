# remix-editor-docs

## Overview
remix-editor v2 のユーザードキュメント。日本語・英語の2言語対応。

## Structure
```
ja/           # 日本語ドキュメント
en/           # 英語ドキュメント
├── 01-getting-started.md
├── 02-ui-overview.md
├── 03-patterns.md
├── 04-curve-editing.md

├── 07-playback.md
├── 08-file-operations.md
├── 09-shortcuts.md

```

## Conventions
- 日英で同じファイル名・構成を維持する
- 画像は各言語フォルダ内の `images/` に配置
- Markdown形式、番号付きファイル名で章順を管理
