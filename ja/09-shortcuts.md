# ショートカット

remix-editor のキーボードショートカット一覧です。

## 基本操作

| キー | 機能 |
|------|------|
| Ctrl + Z | 元に戻す（Undo） |
| Ctrl + Y | やり直す（Redo） |
| Ctrl + Shift + Z | やり直す（Redo） |
| Space | 再生/一時停止 |

## ファイル操作

| キー | 機能 |
|------|------|
| Ctrl + O | インポート |
| Ctrl + Shift + S | エクスポート |

## 編集操作

| キー | 機能 |
|------|------|
| Ctrl + C | コピー |
| Ctrl + X | カット（切り取り） |
| Ctrl + V | ペースト（貼り付け） |
| Delete | 削除 |
| Backspace | 削除 |
| Ctrl + A | 全選択 |
| Escape | 選択解除 |

## ビュー操作

| キー | 機能 |
|------|------|
| Z | 選択範囲にズーム |
| J | 再生位置へジャンプ |

## 波形生成

| キー | 機能 |
|------|------|
| W | 波形生成パネルを開く |
| A | 音声からの波形生成 |

## ピボット

| キー | 機能 |
|------|------|
| P | ピボットを設定/クリア |
| Shift + P | ピボットへジャンプ |
| Ctrl + P | 再生位置をピボットに |

## 再生速度

| キー | 機能 |
|------|------|
| Shift + > | 再生速度を上げる |
| Shift + < | 再生速度を下げる |

## セクションナビゲーション

| キー | 機能 |
|------|------|
| ← | 前のセクションへ |
| → | 次のセクションへ |
| Shift + ← | 前のセクションも選択 |
| Shift + → | 次のセクションも選択 |

## マウス操作

| 操作 | 機能 |
|------|------|
| 左クリック（空白） | ポイント追加 |
| 左クリック（ポイント上） | ポイント選択 |
| 左ドラッグ | ポイント移動 |
| Shift + 左ドラッグ | 範囲選択（時間範囲） |
| Ctrl + 左ドラッグ | 矩形選択 |
| 右ドラッグ | ビューポートをパン（移動） |
| マウスホイール | ズームイン/アウト |
| 選択範囲内ドラッグ | 選択ポイント一括移動 |
| ハンドルドラッグ | 選択範囲リサイズ |
| Alt + ハンドルドラッグ | 対称リサイズ |
| 再生ヘッド領域ドラッグ | 再生位置を変更 |

## 選択範囲リサイズの詳細

| ハンドル | 動作 |
|----------|------|
| 左辺/右辺 | 時間方向にスケール |
| 上辺/下辺 | 値方向に比例スケール |
| 四隅 | 傾斜を付けてスケール（台形変形） |
| Alt + 上下辺 | 中心を維持して両端からスケール |
| Alt + 四隅 | 反対側も同時に調整（対称スケール） |

## ショートカットのカスタマイズ

ショートカットは設定画面からカスタマイズできます。

### 設定画面を開く

![ショートカット設定](./images/12-shortcuts-settings.png)

1. **設定** を開く
2. **ショートカット** タブを選択

### キーの変更

1. 変更したい項目の「変更」ボタンをクリック
2. 新しいキーを押す
3. 「保存」をクリック

### デフォルトに戻す

各項目の「リセット」ボタン、または「すべてデフォルトに戻す」で初期設定に戻せます。

### 設定のエクスポート/インポート

ショートカット設定を JSON ファイルとして保存・読み込みできます。

1. 「エクスポート」でファイルに保存
2. 「インポート」でファイルから読み込み

別の環境に設定を移行する際に便利です。

## 注意事項

### 入力フォーカス中

テキスト入力中（パターン名の編集、セクションの編集など）は、グローバルショートカットが無効になります。

**例外**: Escape キーは入力を終了して通常モードに戻ります。

### モーダル表示中

モーダルダイアログが開いている間は、Escape キー以外のグローバルショートカットは無効です。

- **Escape**: モーダルを閉じる

### ブラウザとの競合

一部のショートカットはブラウザの機能と競合する場合があります。

| キー | ブラウザ機能 | remix-editor での動作 |
|------|--------------|----------------------|
| Ctrl + S | ページ保存 | 無効化（エクスポートを使用） |
| Ctrl + P | 印刷 | ピボット機能に割り当て |
| F5 | ページ更新 | 通常通り機能 |

**ヒント**: 誤ってページを更新しても、自動保存されたデータは復元されます。
