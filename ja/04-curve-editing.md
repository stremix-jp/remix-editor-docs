# カーブ編集

カーブエディタは remix-editor のメイン編集エリアです。時間軸に沿った値の変化をポイントで定義します。

## ポイント操作

### ポイントを追加する

- カーブエディタの空いている場所を**左クリック**
- クリックした位置にポイントが追加されます

### ポイントを選択する

- ポイント上で**左クリック**
- 選択されたポイントは強調表示されます

### ポイントを移動する

1. ポイントを選択
2. **ドラッグ**して移動
3. マウスを離して確定

**注意**: 始点（time=0）と終点（time=maxTime）は Y 座標（値）のみ変更可能です。

### ポイントを削除する

1. ポイントを選択
2. **Delete** キーまたは **Backspace** キーを押す

**注意**: 始点と終点は削除できません。

### ポイントの座標を数値で編集する

1. ポイントを選択
2. 右パネルの **ポイント編集** で Time と Value を入力
3. Enter キーで確定

## 選択範囲操作

### 範囲選択

1. **Shift キーを押しながら**ドラッグ
2. 時間範囲が作成されます
3. 範囲内のポイントがすべて選択されます

### 矩形選択

1. **Ctrl キーを押しながら**ドラッグ
2. 矩形の範囲が作成されます
3. 範囲内のポイントがすべて選択されます

### 全選択

- **Ctrl + A** ですべてのポイントを選択

### 選択解除

- **Escape** キーで選択を解除
- 空白部分をクリックでも解除されます

### 選択範囲の移動

1. 範囲を選択
2. 選択範囲内を**ドラッグ**
3. 選択されたポイントが一括で移動します

### 選択範囲のリサイズ

選択範囲のハンドル（辺や角）をドラッグすることで、ポイントを拡大縮小できます。

| ハンドル | 動作 |
|----------|------|
| 左辺/右辺 | 時間方向にスケール |
| 上辺/下辺 | 値方向に比例スケール |
| 四隅 | 傾斜を付けてスケール（台形変形） |

**Alt キーを押しながら**ドラッグすると、中心を維持して対称にスケールします。

### 選択範囲にズーム

1. 範囲を選択
2. **Z** キーを押す
3. ビューポートが選択範囲に合わせてズームします

## ビューポート操作

### ズームイン/アウト

- **マウスホイール**を回す
- 上: ズームイン
- 下: ズームアウト

### パン（移動）

- **右クリックしながらドラッグ**
- ビューポートが移動します

### ビューポートをリセット

- コンテキストメニューから「ビューをリセット」を選択
- 全体が表示される状態に戻ります

### 再生位置へジャンプ

- **J** キーを押す
- ビューポートが現在の再生位置に移動します

## クリップボード操作

### コピー

1. ポイントまたは範囲を選択
2. **Ctrl + C**
3. 選択されたポイントがクリップボードにコピーされます

### カット

1. ポイントまたは範囲を選択
2. **Ctrl + X**
3. 選択されたポイントがクリップボードにコピーされ、元の位置から削除されます

### ペースト

1. **Ctrl + V**
2. クリップボードのポイントが**現在の再生位置**を基準に貼り付けられます

**注意**: ペースト位置に既存のポイントがある場合、始点・終点以外は上書きされます。

## 波形生成

選択範囲に自動的に波形パターンを生成できます。

### 波形生成パネルを開く

1. 範囲を選択
2. **W** キーを押す、または右パネルの「波形生成」を選択

### 対応波形

#### サイン波

- なめらかな周期的変化
- パラメータ: 振幅、周波数、位相、オフセット、打点間隔

#### 三角波

- 直線的な上昇と下降
- パラメータ: 振幅、周波数、位相、オフセット、打点間隔、ピーク位置

#### 矩形波

- ON/OFF の切り替え
- パラメータ: 振幅、周波数、位相、オフセット、打点間隔

#### ランダム波

- ランダムな値の変化
- パラメータ: 振幅、周波数、位相、オフセット、打点間隔

### パラメータ説明

| パラメータ | 説明 |
|------------|------|
| 振幅 | 波の高さ（maxValue に対する割合） |
| 周波数 | 1秒あたりの波の数 |
| 位相 | 波の開始位置のオフセット |
| オフセット | 波全体の上下位置 |
| 打点間隔 | ポイントを打つ間隔（秒） |
| ピーク位置 | 三角波のピークの位置（0〜1） |

### 周波数実時間モード

- 有効にすると、周波数の値が「選択範囲内の波の数」ではなく「1秒あたりの波の数」になります

## 音声からの波形生成

音声の音量を解析して波形を自動生成できます。

### 音量解析

1. 範囲を選択
2. **A** キーを押す、または「音声からの波形生成」を選択
3. パラメータを設定
4. 「生成」をクリック

パラメータ:
- **閾値**: この音量以下は 0 として扱う
- **打点間隔**: ポイントを打つ間隔

### 別音声ファイルの使用

- 「別ファイルを使用」で現在の音声とは別の音声ファイルを解析できます
- 例: 効果音トラックを解析して振動パターンを生成

## 量子化（クォンタイゼーション）

ポイントをグリッドにスナップさせる機能です。

### 時間量子化

- パターン設定の「時間ステップ」で設定
- ポイントの時間位置が指定した間隔に揃えられます

### 値量子化

- パターン設定の「値ステップ」で設定
- ポイントの値が指定した間隔に揃えられます

## Undo/Redo

すべての編集操作は取り消し・やり直しが可能です。

- **Ctrl + Z**: 元に戻す（Undo）
- **Ctrl + Y** または **Ctrl + Shift + Z**: やり直す（Redo）

## コンテキストメニュー

カーブエディタ上で**右クリック**するとコンテキストメニューが表示されます。

| 項目 | 機能 |
|------|------|
| 元に戻す | Undo |
| やり直し | Redo |
| コピー | 選択ポイントをコピー |
| 切り取り | 選択ポイントを切り取り |
| 貼り付け | クリップボードから貼り付け |
| 削除 | 選択ポイントを削除 |
| 全選択 | 全ポイントを選択 |
| 波形生成 | 波形生成パネルを開く |
| 選択範囲にズーム | 選択範囲を画面いっぱいに表示 |
| ビューをリセット | 全体表示に戻す |
