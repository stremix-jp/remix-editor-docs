# remix-editor ユーザーガイド

remix-editor は、時間軸に沿った値の変化（カーブ）を視覚的に編集し、対応デバイスを制御するための Web アプリケーションです。

10. [FAQ](./10-faq.md) - よくある質問

## 動作環境

- モダンブラウザ（Chrome, Firefox, Edge, Safari）
- デバイス制御には [Intiface Central](https://intiface.com/central/) が必要
