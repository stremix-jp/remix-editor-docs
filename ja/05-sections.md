# セクション

セクションは音声の区間データです。CSV ファイルからインポートし、セクションの属性値から波形を自動生成できます。

## セクションとは

セクションは、音声の特定の時間範囲に関連付けられたデータです。

**例**: 音声の 0:30〜0:45 に「強度: 10」というセクション

これにより、音声の内容に合わせた制御パターンを効率的に作成できます。

## セクションのインポート

### CSV ファイル形式

セクション CSV には以下の列が必要です：

| 列名 | 必須 | 説明 |
|------|------|------|
| start_time | ○ | セクションの開始時間（秒） |
| end_time | ○ | セクションの終了時間（秒） |
| （その他） | × | 任意の属性列 |

**CSV 例**:
```csv
start_time,end_time,intensity_params,note
0,10,5,イントロ
10,30,10,サビ
30,45,15,最高潮
45,60,5,アウトロ
```

**注**: 波形生成に使用する列は `_params` 接尾辞が必要です（詳細は後述）。

### インポート方法

1. **ファイル > セクション CSV インポート** を選択
2. CSV ファイルを選択
3. プレビューを確認
4. 「インポート」をクリック

ドラッグ＆ドロップでもインポートできます。

## セクションパネル

インポートしたセクションはセクションパネルに表示されます。

### 表示モード

- **現在のセクションのみ**: 再生位置にあるセクションの詳細を表示
- **全セクション表示**: すべてのセクションを一覧表示

### セクションの編集

1. セルをクリックして編集モードに入る
2. 値を入力
3. **Enter** で確定、**Escape** でキャンセル

複数行のテキストを入力する場合は **Ctrl + Enter** で改行できます。

## セクションナビゲーション

キーボードでセクション間を移動できます。

| キー | 動作 |
|------|------|
| ← | 前のセクションへ移動 |
| → | 次のセクションへ移動 |
| Shift + ← | 選択範囲を前に拡張 |
| Shift + → | 選択範囲を後に拡張 |

セクションを移動すると、再生位置もそのセクションの開始位置に移動します。

## セクションを選択範囲に設定

1. セクションパネルでセクションを選択
2. 「選択範囲に設定」ボタンをクリック
3. カーブエディタの選択範囲がセクションの時間範囲に設定されます

これにより、セクション単位での波形生成が容易になります。

## ギャップセクション

セクション間に空白がある場合、自動的に「ギャップセクション」が生成されます。

- 表示: グレー背景、値は「-」
- 編集: 不可
- 用途: セクション間の空白を可視化

## セクションからの波形生成

セクションの属性値から自動的に波形を生成できます。

### 波形生成可能な列

**重要**: 波形生成機能は、列名が `_params` で終わる列でのみ使用できます。

**例**:
- `intensity_params` ✓ 波形生成可能
- `speed_params` ✓ 波形生成可能
- `intensity` ✗ 波形生成不可（テキスト列として扱われる）
- `note` ✗ 波形生成不可

CSV を作成する際は、波形生成に使用したい列には必ず `_params` 接尾辞を付けてください。

```csv
start_time,end_time,intensity_params,note
0,10,5,イントロ
10,30,10,サビ
30,45,15,最高潮
45,60,5,アウトロ
```

### 生成ダイアログを開く

1. セクションパネルで列ヘッダーの「生成」ボタンをクリック
2. または、セクションを選択して右クリック > 「波形生成」

### 生成モード

#### 三角波生成

セクションの強度値に基づいて三角波を生成します。

- 単一値（例: `10`）: その強度で三角波を生成
- 範囲値（例: `10->30`）: 開始値から終了値へ変化する三角波を生成

#### 直線波形生成

セクションの強度値に基づいて直線を生成します。

- 単一値: フラットな線を生成
- 範囲値: 傾斜した線を生成

### テキストルール

属性値のテキストを解釈するルールを設定できます。

| ルール | 説明 | 例 |
|--------|------|-----|
| valueRange | 範囲指定の形式 | `10->30` |
| waveformToggle | 波形タイプの切り替え | `triangle`, `line` |
| signDirection | 正負の方向 | `+`, `-` |

### 傾き制御

生成する波形の傾きを制限できます。

- **minSlope**: 最小傾き（これ以下の傾斜は許可しない）
- **maxSlope**: 最大傾き（これ以上の傾斜は許可しない）

急激な変化を避けたい場合に使用します。

### プリセット

波形生成の設定をプリセットとして保存できます。

1. 設定を調整
2. 「プリセット保存」をクリック
3. プリセット名を入力

保存したプリセットは次回から選択できます。

### 列ごとのプリセット

各列に対して個別にプリセットを記憶できます。

- 「intensity」列にはプリセット A
- 「speed」列にはプリセット B

といった使い分けが可能です。

## セクショングループ化

連続するセクションが同じ設定・強度を持つ場合、自動的にグループ化されます。

- グループ単位で三角波の位相が継続
- 空白セクションでグループが分割
- グループ間で位相がリセット

これにより、自然な波形の連続性が保たれます。

## セクションのエクスポート

編集したセクションデータを CSV として保存できます。

1. **ファイル > セクション CSV エクスポート** を選択
2. ファイル名を指定して保存

ファイル名形式: `{音声ファイル名}.{タイムスタンプ}.sections.csv`

## パラメータの書式

`_params` 列に入力する値の書式について説明します。

### 基本形式

```
{強度 0-100}{テキスト}
```

**例**:
- `30` - 強度 30
- `30全体` - 強度 30、テキスト「全体」
- `70根元` - 強度 70、テキスト「根元」

### 変化する値

```
{開始値}->{終了値}{テキスト}
```

**例**:
- `10->30` - 強度が 10 から 30 へ変化
- `0->50全体` - 強度が 0 から 50 へ変化、テキスト「全体」

### テキストの用途

テキスト部分は「テキストルール」と組み合わせて使用します。

**リニア（ペニス）プリセットの例**:
| テキスト | 動作範囲 |
|----------|----------|
| 全体 | 0〜100% |
| 先端 | 70〜100% |
| 軸部 | 30〜70% |
| 根元 | 0〜30% |

**回転プリセットの例**:
| テキスト | 動作 |
|----------|------|
| 右回転 | 正の方向に回転 |
| 左回転 | 負の方向に回転 |

## デバイス別の制作Tips

### リニアデバイス（A10ピストン、The Handy など）

- **シンプルな波形を使う**: 三角波など上下運動が基本
- **複雑な形状は避ける**: 複雑な波形はデバイスの動きがカクつく原因になります
- **待機位置**: 初期位置や待機位置は上端（値=最大値）を推奨

### バイブレーションデバイス（アナルプラグなど）

- **待機位置**: 初期位置や待機位置は下端（値=0）を推奨
- **三角波/サイン波**: ピストン的な動きを表現する場合に使用

### 回転デバイス（乳首責めマシンなど）

- **負の値を許可**: 回転方向を切り替える場合は「負の値を許可」を有効に
- **正の値=右回転、負の値=左回転**
- **左右別々の制御**: 乳首攻めがメインの場合、左右で別々の波形を設定することで臨場感が増します
