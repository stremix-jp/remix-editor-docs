# はじめに

このガイドでは、remix-editor の基本的な使い方を説明します。

## アプリケーションを開く

1. ブラウザで remix-editor の URL にアクセスします
2. 初回起動時は空のパターンが 1 つ作成された状態で開始します

## 基本的なワークフロー

### 1. 音声ファイルを読み込む

1. **フッター**の音声エリアに音声ファイルをドラッグ＆ドロップ
2. または、クリックしてファイル選択ダイアログから選択

対応形式: MP3, WAV, M4A, OGG, FLAC, AAC

### 2. パターンを作成する

1. 左パネルの **パターンリスト** で「+」ボタンをクリック
2. 新しいパターンが作成され、自動的に選択されます
3. パターン名をダブルクリックして名前を変更できます

### 3. カーブを編集する

1. **カーブエディタ**（中央のキャンバス）でクリックしてポイントを追加
2. ポイントをドラッグして移動
3. 複数のポイントを選択して一括編集も可能

詳しくは [カーブ編集](./04-curve-editing.md) を参照してください。

### 4. デバイスを接続する（オプション）

1. [Intiface Central](https://intiface.com/central/) を起動
2. remix-editor の **デバイスパネル** で「接続」をクリック
3. デバイスをスキャンして接続

詳しくは [デバイス接続](./06-devices.md) を参照してください。

### 5. 再生してプレビュー

1. スペースキーまたは再生ボタンで再生開始
2. デバイスが接続されている場合、リアルタイムで制御されます
3. 再生速度の変更も可能

### 6. ファイルを保存する

1. メニューから **ファイル > Remixを保存...** を選択
2. Remix JSON 形式で保存されます

詳しくは [ファイル操作](./08-file-operations.md) を参照してください。
