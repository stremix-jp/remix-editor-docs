# 再生操作

remix-editor では音声を再生しながらパターンをプレビューできます。デバイスが接続されている場合、リアルタイムで制御されます。

## 音声ファイルの読み込み

### ドラッグ＆ドロップ

1. 音声ファイルをフッターの音声エリアにドラッグ
2. ドロップして読み込み

### ファイル選択

1. 音声エリアをクリック
2. ファイル選択ダイアログから選択

### 対応形式

| 形式 | 拡張子 |
|------|--------|
| MP3 | .mp3 |
| WAV | .wav |
| M4A | .m4a |
| OGG | .ogg |
| FLAC | .flac |
| AAC | .aac |

**ファイルサイズ制限**: 最大 100MB

### 自動復元

読み込んだ音声ファイルはブラウザに保存され、次回起動時に自動的に復元されます。

## 再生コントロール

### 再生/一時停止

- **Space** キーまたは再生ボタン
- 再生中に押すと一時停止
- 一時停止中に押すと再生再開

### 停止

- 停止ボタンをクリック
- 再生が停止し、再生位置が先頭に戻ります

### シーク（再生位置の移動）

- 音声波形をクリック
- カーブエディタの再生ヘッド領域をドラッグ
- 任意の位置に移動できます

## 再生速度

### 速度変更

フッターの速度セレクタで変更できます。

| 速度 | 用途 |
|------|------|
| 0.25x | 細かい確認 |
| 0.5x | スロー再生 |
| 1x | 通常速度 |
| 1.5x | やや早め |
| 2x | 2倍速 |
| 4x | 高速確認 |
| 8x | 超高速確認 |

### ショートカット

- **Shift + >**: 速度アップ（循環）
- **Shift + <**: 速度ダウン（循環）

最大/最小で折り返します。

## 音量

### 音量調整

- 音量スライダーで 0% 〜 100% の範囲で調整
- ミュートボタンで即座に消音

## リピート再生

### リピートモード

リピートボタンをクリックすると、ループ再生が有効になります。

### ループ範囲

| 条件 | ループ範囲 |
|------|------------|
| 選択範囲あり | 選択範囲 |
| 選択範囲なし | 現在のビューポート範囲 |

リピート中は、終端に達すると自動的に開始位置に戻ります。

## ピボット機能

ピボットは「お気に入りの位置」のようなものです。頻繁に戻る位置を記憶しておけます。

### ピボットを設定

- **P** キーを押す
- 現在の再生位置がピボットとして保存されます
- 再度 **P** を押すとクリア（トグル動作）

### ピボットへジャンプ

- **Shift + P**
- ビューポートがピボット位置に移動します

### 再生位置をピボットに

- **Ctrl + P**
- 再生位置がピボット位置に移動します

### 永続化

ピボット位置は自動的に保存され、次回起動時にも維持されます。

## 再生位置操作

### 再生位置へジャンプ

- **J** キー
- ビューポートが現在の再生位置に移動します
- 編集中に再生位置を見失った時に便利

### 再生ヘッドのドラッグ

カーブエディタ上部の再生ヘッド領域をドラッグして再生位置を変更できます。

### playheadOffset（再生位置オフセット）

音声とカーブエディタの時間にオフセットを設定できます。

- 範囲: -1秒 〜 +1秒
- 用途: デバイスの応答遅延を補正

設定 > 音声設定 で変更できます。

## 選択範囲再生

### 選択範囲の先頭から再生

1. カーブエディタで範囲を選択
2. フッターの「先頭から再生」ボタンをクリック
3. 選択範囲の開始位置から再生が開始されます

### セクションからの再生

1. セクションパネルでセクションを選択
2. 「選択範囲に設定」でカーブエディタの選択範囲に反映
3. 「先頭から再生」で再生

## 背景波形表示

カーブエディタの背景に音声の波形を表示できます。

### 設定

1. 設定 > 音声設定 を開く
2. 「背景波形を表示」を有効化

### 調整項目

| 設定 | 説明 | 範囲 |
|------|------|------|
| 波形倍率 | 波形の高さ | 10% 〜 100% |
| 波形サンプル数 | 波形の解像度 | 50 〜 4000 |

サンプル数を増やすと詳細になりますが、処理が重くなります。

## スロットル制御

再生中にリアルタイムで値を入力できる機能です。

### 有効化

1. 設定 > 詳細設定 を開く
2. 「スロットル制御を有効化」をオン

### 使い方

1. 再生を開始
2. 縦スライダーを操作
3. 操作した値がリアルタイムでポイントとして記録されます

ライブ入力に便利です。

## デバイスとの同期

再生中、パターンの値は自動的に接続されたデバイスに送信されます。

### 送信タイミング

- 再生中: 約 60fps でデバイスを更新
- 一時停止中: デバイスは現在位置の値を維持

### アクチュエータータイプによる違い

| タイプ | 動作 |
|--------|------|
| Vibrate | 即時制御（値が直接強度になる） |
| Rotate | 即時制御（正負で回転方向が変わる） |
| Linear | 位置ベース（ドラッグ終了時に移動） |

## トラブルシューティング

### 音声が再生されない

1. ブラウザのミュートを確認
2. 音量スライダーを確認
3. 対応形式のファイルか確認

### 再生がカクつく

1. 背景波形のサンプル数を減らす
2. 他のタブを閉じる
3. ブラウザを再起動

### デバイスと音声がずれる

1. playheadOffset を調整
2. デバイスの遅延特性を確認
